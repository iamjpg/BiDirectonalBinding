BiDirectionalBinding.js

A super simple two-way binding plugin written in vanilla Javascript.

Form

Enter values in the form below and see the bound javascript object change with it. Example: Enter any value you like in the First Name field. After, in the console to the right, type data.firstName and press Enter.


Browser JS Console

In the console below, enter something like data.firstName = "John" and press enter to see the form value change.

This plugin was written by JP Given and uses the awesome WatchJS library written by melanke.
